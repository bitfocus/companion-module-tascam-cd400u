## Tascam CD-400u Media Player

This module will control the Tascam DA-6400.

- [Tascam CD-400U Page](https://tascam.com/us/product/cd-400u/)

## Configuration
Enter the IP address, port and password of the recorder.

## Actions
- **Stop** 
- **Play** 
- **Pause** 
- **Search** 
- **Eject** 
- **Skip**
- **Resume Play Mode**
- **Incremental Playback**
- **Clear**
- **Remote/Local Select**
- **Play Mode Select**
- **Current Track Time**
- **Device Select**
- **Play Area Select**
- **Enter**
- **Back**

## Variables
- **Track Number**
- **Error Status**
- **Caution Status**
- **Device Status**
